Sobre Mí

Soy un analista de datos y desarrollador web junior con certificaciones en Google Data Analytics y BigQuery (Coursera), Desarrollo Web (HTML, CSS, JavaScript) de Coderhouse, y Power BI de Santander Open Academy. Busco oportunidades remotas o híbridas en CABA, Argentina, para crear soluciones impactantes con datos y desarrollo web.
